# Constants
## Z-Index

###### Higher = in front

**-5**
- Backerground TileMap Layer

**-4**
- Background TileMap Layer

**-3**
- BackMidground TileMap Layer *(for decals on Background)*

**-2**
- Midground TileMap Layer

**-1**
- Back Arm

**+0**
- Player

**+1**
- Layer0 TileMap Layer

**+3**
- Front Arm

**+4**
- Foreground TileMap Layer
- Flag

**+5**
- Secret zone fog of war (Layer0)

**+6**
- ????

**+7**
- Back Arm Particles

**+8**
- Front Arm Particles

**+100**
- Cursor *(Sprite)*
